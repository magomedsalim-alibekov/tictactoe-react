Стандартная игра в крестики нолики, которую можно увидеть в документации по React. Ничего удивительного. 
Но если вы хотите поиграть в нее, она просто так не запуститься. Смотрите ниже:

Заметка: для начала у час должен быть установле Node.js. Если не установлен, скачивайте с официального сайта: https://nodejs.org/

Инструкция:
1. Скачайте файлы из репозитория, далее распакуйте ее в любой папке.
2. Далее откройте терминал. В терминале вы должны вести путь к ранее скачанным файлам. 
Сделать это можно вот так:
`` cd C:Users/путь/к/проекту/ ``
3. После того, как вы оказались в папке проекта, начните вести команду ``npm install`` - это установит все зависимости для корректной работы сайта
4. Когда все скачается, запустите проект: ``npm start``
5. Сайт должен открыться автоматически на вашем браузере и вы можете спокойно попробовать поиграть в эту игру.
5.1. Если сайт не запустился автоматом, в терминале будет написано что то похожее на это:

"You can now view react.dev in the browser.

  Local:            http://{your local host}
  On Your Network:  http://{your own network}

Note that the development build is not optimized.
To create a production build, use npm run build."

Просто нажмите на первую ссылку выше (используйте CTRL/CMD + левую кнопку мыши), чтобы открыть сайт.

Готово! Если все сделаете правильно, вы сможете увидеть 9 квадратиков в левом верхнем углу и вы сможете поиграть в крестики нолики со своим другом!
